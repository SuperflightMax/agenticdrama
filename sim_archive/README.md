# Simulation Starter Pack

Набор стартовых файлов для симуляции персонажей через **пространство + ситуацию + состояние**, без прямого сценария поведения.

## Что внутри

- `meta/model_notes.md` — краткая идея модели
- `templates/scene.template.yaml` — шаблон пространства
- `templates/situation.template.yaml` — шаблон ситуации
- `templates/run.template.yaml` — шаблон запуска отрезка симуляции
- `scenes/home_shared.yaml` — дом / совместное жильё
- `scenes/work_office.yaml` — работа / офис одного из персонажей
- `scenes/relax_outdoor.yaml` — отдых / парк / прогулка
- `situations/need_to_decide.yaml` — надо принять решение
- `situations/something_is_off.yaml` — что-то не так
- `situations/mismatch_expectations.yaml` — несовпадение ожиданий

## Идея

Поведение персонажам **не задаётся напрямую**.
Мы задаём:

1. пространство,
2. ситуацию,
3. текущее состояние персонажей,
4. состав действующих лиц.

А дальше модель сама разворачивает паттерны поведения.

## Как использовать

Под каждое место / тип дропа можно делать отдельную кампанию:

- `home_*`
- `work_*`
- `relax_*`

Кампания тут понимается как **набор продолжаемых отрезков симуляции в одном и том же пространстве**.

Пока без переходов между пространствами внутри одной линии.
Это можно добавить позже отдельным слоем.
