# WORLD.md — Campaign World State

## Human-like memory

Память игроков не является полным архивом событий.

Она формируется автоматически из состояния и его изменения.
Игрок не может надёжно "решить запомнить" что-то по воле.
Зато сильные или длительные изменения состояния создают сильный отпечаток.

### Что формирует память

Память формируется из:
- контекста ситуации (кто, где, что происходило)
- изменения состояния
- пережитого ощущения
- сделанной в тот момент оценки / вывода

### Главное правило

Сила памяти зависит не только от пика реакции,
а от накопленного изменения состояния во времени.

То есть:
- резкий сильный всплеск создаёт сильную память
- длительное давление / повторяющееся переживание тоже создаёт сильную память

Пример:
- не только "меня резко напугали"
- но и "меня долго морозили, игнорировали, напрягали" → тоже сильный след

### Что память сохраняет

Память может включать:
- признаки ситуации
- фразы
- ключевые факты
- телесное / эмоциональное ощущение
- сделанный тогда вывод

Например:
- "p2 меня кинул"
- "с ним небезопасно"
- "рядом с ним мне спокойно"
- "в сарае втроём тяжело дышать"

Память не обязана быть объективной.
Она хранит пережитую версию мира.

### Слабые и сильные следы

- слабые следы быстро теряют детализацию и исчезают
- сильные следы живут дольше и устойчивее
- сильные следы могут искажаться, но не исчезают быстро

### Memory Imprint Format

```json
{
  "id": "imprint_001",
  "cues": ["river", "p2", "promise", "food"],
  "state_signature": ["hope", "tension", "hurt", "drop in trust"],
  "evaluation": "p2 betrayed me",
  "fragments": ["I trusted him", "he gave it to p1"],
  "strength": 82,
  "valence": "negative",
  "clarity": "vivid"
}
```

### Memory strength

memory strength зависит от accumulated state change.

- сильный резкий сдвиг → большой imprint
- длительное накапливающееся изменение → тоже большой imprint
- слабая ровная ситуация → imprint почти не формируется

### Memory trigger

Когда текущая ситуация похожа на ранее записанный imprint,
агент НЕ обязан явно "вспоминать" прошлое.

Вместо этого imprint может сработать как:
- мгновенная оценка
- ощущение знакомости
- тревога
- симпатия
- ощущение давления
- чувство безопасности

Это не обязано сопровождаться объяснением.

Примеры:
- "я не знаю почему, но мне это не нравится"
- "мне с ним спокойно"
- "это уже плохо пахнет"
- "что-то тут знакомо и херово"

### Recall / memory drift

Если imprint всё же явно вспоминается,
он не читается как неизменный лог.
Он пересобирается под влиянием текущего состояния.

- recall в сильном состоянии может немного усилить или исказить память
- recall в более спокойном состоянии искажает меньше
- каждая эмоционально заряженная "перепроживка" может усилить strength и evaluation

Примеры искажения:
- "он меня подвёл" → "он меня кинул" → "он всегда так делает"
- "было неприятно" → "рядом с ним всегда тревожно"

Искажение небольшое, в сторону текущего состояния.

### Forgetting / fading

- слабые imprintы теряют ясность, теряют детали, исчезают
- сильные imprintы живут дольше, могут терять точность но сохраняют вес

## State-driven perception

### State → memory activation

Состояние определяет какие imprints активируются:

- в негативном состоянии (stress > 70, mood < 40): активируются негативные imprints
- в позитивном состоянии (stress < 40, mood > 60): активируются позитивные imprints

Не все воспоминания доступны одновременно. Память — это набор отпечатков, часть которых "поднимается" в зависимости от состояния.

### World feel

Мир ощущается по-разному в зависимости от состояния:

- **tense / heavy / hostile** — в негативном состоянии
- **calm / open / manageable** — в позитивном состоянии

### Attention bias

В негативном состоянии:
- внимание сужается к угрозам и негативу
- позитивные детали игнорируются или обесцениваются

В позитивном состоянии:
- внимание к возможностям и безопасности
- негатив сглаживается или игнорируется

### Reinterpretation

Один и тот же факт может означать разное:

- в негативе: нейтральное → фальшивое / манипулятивное
- в позитиве: негативное → незначительное / объяснимое

Меняется не факт, а его смысл.

### State inertia

Состояние стремится сохраняться:
- негатив ищет подтверждение
- позитив сглаживает негатив

### Resistance to change

Смена состояния требует достаточного "веса" события:
- в сильном негативе позитивные сигналы обесцениваются
- в сильном позитиве негативные сигналы сглаживаются
