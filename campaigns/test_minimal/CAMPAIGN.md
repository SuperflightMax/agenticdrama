# CAMPAIGN.md

## Name
test_minimal

## Purpose
Minimal smoke test for the full pipeline:
- runtime activation,
- Run DM packet assembly,
- player packets,
- perception -> response -> consequence -> logging.

This is intentionally small.
It should show whether the system works at all before running richer campaigns.

## Active cast
- `oleg` — slightly impulsive, wants quick relief and tends to reach first
- `mira` — cautious, prefers low-conflict solutions and notices discomfort quickly

## Main pressures
- mild hunger
- limited shelter space
- very limited food resource

## Important deviations
Use default baseline rules.
No special campaign-only mechanic beyond simple scarcity and shelter pressure.
