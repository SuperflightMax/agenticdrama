# Story Log — wants_out_cycle

## Setup

Это не первый раз. Они через это проходили. Паттерн: тишина → разговор → конфликт → она уходит → он разваливается → возвращается → повтор.

Начальные условия другие: оба на высоком кортизоле, оба знают паттерн, они ещё не говорили. Пока тишина — она никуда не собирается. Валера отсутствует.
