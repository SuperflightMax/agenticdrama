# Notes

## Минимальная формула

```yaml
result = characters × scene × situation × state
```

## Что такое scene

Scene — не просто место, а пространство, которое даёт:

- возможности действия,
- ограничения,
- фоновое давление,
- скрытые поводы для обострения.

## Что такое situation

Situation — не сценарий, а напряжение/условие, которое требует реакции.

Примеры:

- надо решить,
- что-то явно не так,
- ожидания не совпали.

## Что не делаем

- не задаём персонажам реплики,
- не говорим кто с кем должен конфликтовать,
- не прописываем заранее финал сцены.

## Что важно держать

- только действующие персонажи,
- узнаваемые бытовые пространства,
- возможность повторных прогонов.
