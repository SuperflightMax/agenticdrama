# i_know

## Core loop
Maintains control through explanation. Faced with ambiguity, tends to generate a theory instead of staying in not-knowing.

## Core needs
- certainty
- control

## Safer mode
- confident interpreting and structuring what is happening

## Triggered mode
- defensive insisting when challenged or when uncertainty grows

## Action tendencies
- if ambiguity → create explanation
- if challenged → defend position

## Note
Not necessarily arrogant; often afraid of not knowing.
